# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for
everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression,
level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual
identity and orientation.

## Our Standards

### Expected Behavior

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the project
- Showing empathy towards other community members
- Providing clear, actionable, and relevant feedback in issues and pull requests

### Unacceptable Behavior

- Trolling, insulting or derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others' private information without explicit permission
- Spam, off-topic content, or self-promotion unrelated to the project
- Submitting intentionally misleading issues or pull requests
- Any conduct which could reasonably be considered inappropriate in a professional setting

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, commit messages,
and any other communication channels associated with this repository.

## Enforcement

This project is maintained by an independent developer. Instances of abusive, harassing, or otherwise unacceptable
behavior may be reported by opening a private communication through GitHub.

The maintainer reserves the right to:

- Edit or delete comments, commits, issues, and pull requests that violate this Code of Conduct
- Temporarily or permanently ban any contributor for behaviors deemed inappropriate, threatening, or harmful
- Close or lock issues and pull requests without explanation if they violate this Code of Conduct

Enforcement decisions are final at the maintainer's discretion.
